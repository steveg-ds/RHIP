# Design Spec: Reduce Code Duplication in RHIP Data Exploration

**Date:** 2026-06-16  
**Author:** Antigravity  
**Status:** Draft  

---

## 1. Goal

Reduce code duplication between `urban-science/Data Exploration.ipynb` and `rural-detention/Data Exploration.ipynb` by extracting shared ACS data fetching and calculation logic into a common Python module under `utils/`. Use Pydantic for configuration validation.

---

## 2. Architecture & Design Details

### 2.1 Shared Module: `utils/data_loader.py`

Create a new module `utils/data_loader.py` containing:
- **`CensusConfig` (Pydantic Model):** Validates parameters `year`, `states`, and `api_key`.
- **`CensusDataLoader` (Class):**
  - Batch fetches all 39 required ACS variables in a single Census API call.
  - Caches the raw result internally to avoid redundant network requests.
  - Provides helper methods to calculate specific metrics (`PovertyRate`, `CostBurden`, `HousingQuality`, `LessThanHS`, `PercentMinority`, `Gini`).
  - Provides a master method `get_merged_dataset(ruca_df)` that aggregates all metrics and merges them with RUCA codes.

### 2.2 Variables Mapping

The loader will query the following variables in one batch:

| Category | Variables |
| --- | --- |
| **Rent Burden** | `B25070_001E`, `B25070_007E`, `B25070_008E`, `B25070_009E`, `B25070_010E` |
| **Owner Burden** | `B25091_001E`, `B25091_008E`, `B25091_009E`, `B25091_010E`, `B25091_011E`, `B25091_019E`, `B25091_020E`, `B25091_021E`, `B25091_022E` |
| **Poverty** | `B17001_001E`, `B17001_002E` |
| **Housing Quality** | `B25047_001E`, `B25047_003E`, `B25051_001E`, `B25051_003E` |
| **Education** | `B15003_001E` and `B15003_002E` through `B15003_016E` |
| **Demographics** | `B03002_001E`, `B03002_003E` |
| **Gini Index** | `B19083_001E` |

### 2.3 Calculations

- **`CostBurden`:** `(sum(B25070_007E..010E) + sum(B25091_008E..011E) + sum(B25091_019E..022E)) / (B25070_001E + B25091_001E)`
- **`PovertyRate`:** `B17001_002E / B17001_001E`
- **`HousingQuality`:** `(B25047_003E + B25051_003E) / B25047_001E`
- **`LessThanHS`:** `sum(B15003_002E..016E) / B15003_001E`
- **`PercentMinority`:** `1 - (B03002_003E / B03002_001E)`
- **`Gini`:** Direct mapping of `B19083_001E`
